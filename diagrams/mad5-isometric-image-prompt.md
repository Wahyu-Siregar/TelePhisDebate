# MAD5 Isometric Image Prompt

## Main Prompt
```text
Create an isometric 3D infographic showing how a MAD5 (5-agent Multi-Agent Debate) phishing detection system works in a Telegram group context.

Scene layout (left to right, top-down isometric grid):
1) Input Zone: Telegram-style chat panel with one suspicious message containing a shortened URL.
2) Stage 1 Zone: Rule-Based Triage module (shield + filter icon) that quickly labels low risk vs escalated cases.
3) Stage 2 Zone: Single-Shot LLM Router module (AI chip icon) that routes uncertain/high-risk content to MAD5.
4) Stage 3 Zone: Central “Debate Chamber” with 5 distinct AI agent pods connected in a pentagon:
   - Detector Agent
   - Critic Agent
   - Defender Agent
   - Fact-Checker Agent
   - Judge Agent
   Show Round 1 as individual outgoing analysis beams from each pod to the center.
   Show Round 2 as cross-deliberation arrows between all agent pods.
5) Aggregation Zone: Weighted Voting Aggregator console with bar weights and vote tally.
6) Output Zone: Final Decision board with 3 outcomes:
   - PHISHING
   - SUSPICIOUS
   - LEGITIMATE
   plus confidence score and recommended action.

Visual style:
- Isometric cyber-security control room
- Clean professional academic infographic
- Distinct modules with clear labels and directional arrows
- Color system: deep navy base, cyan data lines, orange warning accents, green safe indicator, red phishing indicator
- Soft glow effects, crisp vector-like edges, high readability, no clutter
- 16:9 aspect ratio, ultra-detailed, presentation-ready
```

## Short Prompt
```text
Isometric 3D infographic of MAD5 phishing detection pipeline: Telegram message input -> Rule-Based Triage -> Single-Shot LLM Router -> 5-agent debate chamber (Detector, Critic, Defender, Fact-Checker, Judge) with Round 1 independent analysis and Round 2 cross-deliberation -> Weighted Voting Aggregator -> Final decision (PHISHING/SUSPICIOUS/LEGITIMATE + confidence). Clean cybersecurity style, professional, labeled modules, 16:9.
```

blurry text, unreadable labels, low resolution, crowded composition, random extra characters, distorted arrows, broken perspective, childish style, anime style, photoreal humans
